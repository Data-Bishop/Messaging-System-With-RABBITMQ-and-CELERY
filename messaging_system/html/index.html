Welcome to the Messaging System

This system provides various functionalities. Here's how to use them:

Sending an Email

To queue an email, use the following URL format:

https://grateful-antelope-holy.ngrok-free.app/?sendmail=example@email.com

Logging the Current Time

To log the current time, use:

https://grateful-antelope-holy.ngrok-free.app/?talktome

Accessing Logs

To view the logs, visit:

https://grateful-antelope-holy.ngrok-free.app/logs
